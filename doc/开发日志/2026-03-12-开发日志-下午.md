## 🔧 具体工作内容

### 1. 前端 API 调用问题修复

#### 1.1 问题发现

**现象**：
- 后台首页显示所有数据为 0
- 图表空白
- Console 报错：`TypeError: Cannot read properties of undefined (reading 'get')`

**原因分析**：
- `index.vue` 中使用了 `this.$axios.get()`
- 但 `axios` 没有被挂载到 Vue 原型上
- `this.$axios` 是 `undefined`

#### 1.2 修复方案

**修改文件**：`eduswap-ui/src/views/index.vue`

**修改内容**：
```javascript
// 修复前
import * as echarts from 'echarts'

const response = await this.$axios.get('/system/home/stat')

// 修复后
import * as echarts from 'echarts'
import request from '@/utils/request'

const response = await request({
  url: '/system/home/stat',
  method: 'get'
})
```

**修复的方法**：
1. `loadHomeData()` - 加载首页统计数据
2. `loadVisitTrend()` - 加载访问趋势图表
3. `loadUserTypeDistribution()` - 加载用户类型分布

## 📊 技术要点总结

### 1. Axios 使用规范

**若依框架推荐方式**：
```javascript
// 方式 1：使用 request 工具（推荐）
import request from '@/utils/request'
const response = await request({ url, method })

// 方式 2：使用 this.$axios（需要挂载）
// 在 main.js 中添加
Vue.prototype.$axios = axios
```

### 2. 代码重构原则

**本次重构遵循**：
- 单一职责原则：每个方法只负责一个功能
- 开闭原则：便于后续扩展真实数据查询
- 注释规范：详细的 TODO 注释和 SQL 说明
