## 🏗️ 项目架构

### 技术架构
- **后端框架**: Spring Boot 2.5.15 + Spring Security 5.7.14
- **前端框架**: Vue 2 + Element UI
- **数据库**: MySQL + Druid 连接池
- **权限控制**: RBAC 模型 + JWT Token
- **代码生成**: 若依代码生成器

### 模块结构
```
eduswap/
├── eduswap-admin      # 后台管理模块
├── eduswap-framework  # 核心框架模块
├── eduswap-system     # 系统管理模块
├── eduswap-quartz     # 定时任务模块
├── eduswap-generator  # 代码生成模块
├── eduswap-common     # 通用工具模块
└── eduswap-ui         # 前端页面模块
```

### 环境要求

- Maven 3.6+
- MySQL 5.7+
- Redis 5.0+
