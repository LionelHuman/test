### 2. 后端接口开发

#### 2.1 创建控制器

**文件**：`eduswap-system/src/main/java/com/eduswap/system/controller/SysHomeController.java`

**功能**：
- 提供后台首页统计数据接口
- 提供访问趋势接口
- 提供用户类型分布接口

**接口列表**：
```java
@RestController
@RequestMapping("/system/home")
public class SysHomeController {
    
    // 获取首页统计数据
    @GetMapping("/stat")
    public AjaxResult getHomeStat()
    
    // 获取近 7 日访问趋势
    @GetMapping("/visit/trend")
    public AjaxResult getVisitTrend()
    
    // 获取用户类型分布
    @GetMapping("/user/type")
    public AjaxResult getUserType()
}
```

#### 2.2 修复编译错误

**问题 1**：SysUser 导入路径错误

```java
// 错误
import com.eduswap.system.domain.SysUser;

// 正确
import com.eduswap.common.core.domain.entity.SysUser;
```

**问题 2**：PageHelper 使用错误
```java
// 错误
long userCount = list.getTotal();

// 正确
PageInfo<SysUser> pageInfo = new PageInfo<>(list);
long userCount = pageInfo.getTotal();
```

**解决方案**：
- 修正导入路径
- 使用 PageInfo 包装 List 获取总数

### 4. 前端集成

#### 4.1 修改 index.vue

**文件**：`eduswap-ui/src/views/index.vue`

**修改内容**：
```javascript
// 添加数据属性
data() {
  return {
    userCount: 0,
    visitCount: 0,
    taskCount: 0,
    monthIncome: '¥0'
  }
},

// 添加 API 调用方法
async loadHomeData() {
  const response = await this.$axios.get('/system/home/stat')
  if (response.code === 200 && response.data) {
    const data = response.data
    this.userCount = data.userCount || 0
    this.visitCount = data.visitCount || 0
    this.taskCount = data.taskCount || 0
    this.monthIncome = '¥' + (data.monthIncome || 0).toLocaleString()
  }
}
```

### 5. 问题排查与解决

#### 5.1 端口占用问题

**现象**：
```
Port 8080 is already in use
```

**解决**：
```bash
lsof -ti:8080 | xargs kill -9
```

#### 5.2 权限认证问题

**现象**：
```json
{
  "code": 401,
  "msg": "认证失败，无法访问系统资源"
}
```

**原因**：
- 新添加的权限未生效
- token 中未包含新权限

**解决**：
1. 退出登录
2. 清除浏览器缓存
3. 重新登录

#### 5.3 数据加载失败

**现象**：
- 页面显示"首页数据加载失败"
- 所有数据为 0

**排查步骤**：
1. 检查后端服务是否运行
2. 检查接口路径是否正确
3. 检查权限配置
4. 检查前端 API 调用
5. 检查浏览器缓存
