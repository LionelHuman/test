## 🔧 技术细节

### 文件编码

- **编码格式**：UTF-8
- **换行符**：LF（Mac/Linux）或 CRLF（Windows）
- **缩进**：2 个空格

### Markdown 格式

- **标题**：使用 `#` 标记
- **列表**：使用 `-` 或 `1.`
- **代码**：使用 ` ```language ` 包裹
- **链接**：使用 `[文本](路径)`

### Git 提交

**建议提交信息**：
