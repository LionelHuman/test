## 🔧 问题解决记录

### 问题 1：编译错误

**现象**：找不到符号 `SysUser`

**原因**：导入路径错误

**解决**：
```java
// 错误
import com.eduswap.system.domain.SysUser;

// 正确
import com.eduswap.common.core.domain.entity.SysUser;
```

### 问题 2：PageHelper 使用

**现象**：`list.getTotal()` 方法不存在

**原因**：PageHelper 返回的 List 没有 getTotal() 方法

**解决**：
```java
PageInfo<SysUser> pageInfo = new PageInfo<>(list);
long total = pageInfo.getTotal();
```

### 问题 3：权限不生效

**现象**：接口返回 401 认证失败

**原因**：新添加的权限未更新到 token

**解决**：退出登录 → 清除缓存 → 重新登录
