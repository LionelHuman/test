## 📞 常见问题

### Q1: 每天会生成多篇吗？

**A**: 不会。即使一天多次对话，我也会合并成一篇完整的日志。

### Q2: 周末会生成吗？

**A**: 如果您周末在工作并有开发活动，我会生成；否则不生成。

### Q3: 能否查看之前的日志？

**A**: 可以。打开 `doc/开发日志/` 目录查看所有历史日志。

### Q4: 能否修改已生成的文档？

**A**: 可以。您可以随时编辑任何日志，我会帮您完善。

### Q5: 如果某天不想生成怎么办？

**A**: 告诉我不生成即可，例如："今天不用生成文档"。

### Q6: 文档太长怎么办？

**A**: 可以要求我精简，例如："以后的文档简洁一些"。

### Q7: 如何查看今日日志？

**A**: 
```bash
# Mac
cd doc/开发日志
open $(ls -t *.md | head -1)

# 或在 IDEA 中直接打开最新文件
```
