## 🔧 已修复的问题

### 1. 商品分类下拉框无数据 ✅
- **原因**：前端尝试从字典获取数据，但字典未配置
- **解决**：
  - 在字典管理中创建 `product_class` 字典类型
  - 插入 10 个商品分类数据
  - 前端代码使用 `dicts: ['product_class']` 读取字典

### 2. 发布商品时 user_id 为空 ✅
- **原因**：biz_product 表的 user_id 字段不能为空，前端未传递
- **解决**：
  - 前端获取当前登录用户 ID：`this.$store.state.user.id`
  - 提交时包含 userId 参数
  - 添加登录验证，未登录跳转到登录页

### 3. SCSS 编译错误 ✅
- **原因**：使用了旧的 `/deep/` 语法
- **解决**：改为 `::v-deep` 语法
- **影响文件**：Home.vue, ProductDetail.vue

## 📊 数据库查询语句

### 查询所有字典数据
```sql
SELECT * FROM sys_dict_data 
WHERE dict_type IN ('product_class', 'product_grade') 
ORDER BY dict_type, dict_sort;
```

### 查询商品列表（带分类名称）
```sql
SELECT p.*, c.category_name 
FROM biz_product p
LEFT JOIN biz_product_category c ON p.category_id = c.category_id
ORDER BY p.create_time DESC;
```

### 查询在售商品
```sql
SELECT * FROM biz_product 
WHERE status = '1' AND del_flag = '0'
ORDER BY create_time DESC;
```
