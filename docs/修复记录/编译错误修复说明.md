# 编译错误修复说明

## ❌ 错误原因

**错误信息**：
```
java: 找不到符号
符号：类 SysUser
位置：程序包 com.eduswap.system.domain
```

**原因**：
- `SysUser` 类的实际位置是 `com.eduswap.common.core.domain.entity.SysUser`
- 错误的导入路径：`com.eduswap.system.domain.SysUser` ❌
- 正确的导入路径：`com.eduswap.common.core.domain.entity.SysUser` ✅

**文件**：`eduswap-system/src/main/java/com/eduswap/system/controller/SysHomeController.java`

```java

// 修改后（正确）
```

## 📝 常见编译错误

### 错误 1：找不到符号 - 类 XXX

**原因**：导入路径错误

**解决**：
1. 在 IDEA 中按 `Ctrl + N`（Mac: `Cmd + O`）
2. 搜索类名
3. 查看正确的包路径
4. 修改导入语句

### 错误 2：程序包不存在

**原因**：依赖未下载或 Maven 配置问题

**解决**：
```bash
# 清理并重新下载依赖
mvn clean install -U -DskipTests
```

### 错误 3：编码问题

**原因**：文件编码不一致

**解决**：
1. IDEA 设置 → Editor → File Encodings
2. 全部设置为 UTF-8
3. 重新编译
