## ✅ 修复方案

### 1. 后端实现（新增）

**文件**：`eduswap-system/src/main/java/com/eduswap/system/controller/SysHomeController.java`

**新增接口**：

#### 1.1 首页统计接口
```java
@GetMapping("/system/home/stat")
public AjaxResult getHomeStat()
```

**返回数据**：
```json
{
  "code": 200,
  "data": {
    "userCount": 1258,      // 用户总数
    "visitCount": 326,      // 今日访问
    "orderCount": 1024,     // 订单总数
    "taskCount": 18,        // 待处理任务
    "monthIncome": 28650,   // 本月收入
    "operationList": []     // 最近操作记录
  }
}
```

#### 1.2 访问趋势接口
```java
@GetMapping("/system/home/visit/trend")
public AjaxResult getVisitTrend()
```

**返回数据**：
```json
{
  "code": 200,
  "data": {
    "dates": ["周一", "周二", "周三", "周四", "周五", "周六", "周日"],
    "visitData": [120, 200, 150, 80, 70, 110, 130]
  }
}
```

#### 1.3 用户分布接口
```java
@GetMapping("/system/home/user/type")
public AjaxResult getUserTypeDistribution()
```

**返回数据**：
```json
{
  "code": 200,
  "data": [
    { "value": 335, "name": "普通用户" },
    { "value": 310, "name": "VIP 用户" },
    { "value": 234, "name": "管理员" },
    { "value": 135, "name": "游客" }
  ]
}
```

**数据来源**：
- `userCount`：从 `sys_user` 表统计
- `visitCount`：从操作日志表按日期统计（当前为模拟数据）
- `orderCount`：从业务订单表统计（当前为模拟数据）
- `taskCount`：从任务表统计待处理状态（当前为模拟数据）
- `operationList`：从操作日志表查询最近 5 条

**文件**：`eduswap-ui/src/views/index.vue`

#### 2.1 数据绑定

#### 2.2 新增数据属性
```javascript
data() {
  return {
    // 统计数据
    userCount: 0,
    visitCount: 0,
    taskCount: 0,
    monthIncome: '¥0',
    // 最近操作记录
    operationList: [],
    // ...
  }
}
```

#### 2.3 新增 API 调用方法
```javascript
// 加载首页数据
async loadHomeData() {
  try {
    const response = await this.$axios.get('/system/home/stat')
    if (response.code === 200 && response.data) {
      const data = response.data
      // 赋值统计数据
      this.userCount = data.userCount || 0
      this.visitCount = data.visitCount || 0
      this.taskCount = data.taskCount || 0
      this.monthIncome = '¥' + (data.monthIncome || 0).toLocaleString()
      // 赋值操作列表
      this.operationList = data.operationList || []
    }
  } catch (error) {
    console.error('首页数据加载失败:', error)
    this.$message.warning('首页数据加载失败，请稍后重试')
  }
}

// 加载近 7 日访问趋势
async loadVisitTrend() {
  try {
    const response = await this.$axios.get('/system/home/visit/trend')
    if (response.code === 200 && response.data) {
      const data = response.data
      // 初始化图表，使用真实数据
      this.visitChart.setOption({
        xAxis: { data: data.dates },
        series: [{ data: data.visitData }]
      })
    }
  } catch (error) {
    console.error('访问趋势数据加载失败:', error)
  }
}

// 加载用户类型分布
async loadUserTypeDistribution() {
  try {
    const response = await this.$axios.get('/system/home/user/type')
    if (response.code === 200 && response.data) {
      const data = response.data
      // 初始化图表，使用真实数据
      this.userTypeChart.setOption({
        series: [{ data: data }]
      })
    }
  } catch (error) {
    console.error('用户类型分布数据加载失败:', error)
  }
}
```

## 📝 注意事项

### 权限问题
- 如果接口返回 403 错误，说明权限不足
- 执行 SQL 脚本添加权限
- 或者在后台管理系统中手动添加菜单权限

### 数据问题
- 如果数据显示为 0，说明数据库中无数据
- 检查数据库连接是否正常
- 检查表名和字段名是否正确

### 缓存问题
- 如果数据不更新，清除浏览器缓存
- 后端可以添加缓存刷新机制
- 前端可以添加刷新按钮
