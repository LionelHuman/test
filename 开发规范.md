## 💻 代码规范

### Java 后端规范

#### 1. 命名规范

**类命名**:
```java
// ✅ 正确
public class BizProduct { }
public class SysUserController { }
public class IBizProductService { }

// ❌ 错误
public class bizProduct { }
public class Sysusercontroller { }
```

**方法命名**:
```java
// ✅ 正确
public List<BizProduct> selectBizProductList(BizProduct bizProduct);
public int insertBizProduct(BizProduct bizProduct);
public void deleteBizProductByProductId(Long productId);

// ❌ 错误
public List<BizProduct> getProducts(BizProduct p);
public int addProduct(BizProduct product);
```

**变量命名**:
```java
// ✅ 正确
private Long userId;
private String productName;
private List<BizProduct> productList;

// ❌ 错误
private Long id;
private String name;
private List<BizProduct> list;
```

**常量命名**:
```java
// ✅ 正确
public static final String USER_STATUS = "status";
public static final int MAX_PAGE_SIZE = 100;

// ❌ 错误
public static final String status = "status";
```

#### 2. 注释规范

**类注释**:
```java

 *

 * @date 2026-03-10
 */
```

**方法注释**:

```java
 * @return 商品集合
 */
```

**字段注释**:
```java

```

#### 3. 代码格式

**缩进**: 4 个空格（不使用 Tab）

**行宽**: 不超过 120 字符

**大括号**:
```java
// ✅ 正确
if (condition) {
    // code
}

// ❌ 错误
if (condition) { /* code */ }
```

**空行**:
- 方法之间空 1 行
- 方法内逻辑段落空 1 行
- 类定义后空 1 行

#### 4. 异常处理

**异常捕获**:
```java
// ✅ 正确
try {
    // business logic
} catch (ServiceException e) {
    log.error("业务异常：{}", e.getMessage(), e);
    throw e;
} catch (Exception e) {
    log.error("系统异常", e);
    throw new ServiceException("系统异常");
}

// ❌ 错误
try {
    // business logic
} catch (Exception e) {
    // 不处理或吞掉异常
}
```

**异常抛出**:
```java
// ✅ 正确
if (user == null) {
    throw new ServiceException("用户不存在");
}

// ❌ 错误
if (user == null) {
    return null; // 应该抛出异常
}
```

#### 5. 日志规范

```java
// ✅ 正确
log.info("查询商品列表，参数：{}", JSON.toJSONString(params));
log.debug("商品详情：{}", JSON.toJSONString(product));
log.warn("商品库存不足，productId: {}", productId);
log.error("创建商品失败", e);

// ❌ 错误
System.out.println("查询商品");
log.info("查询商品：" + product.toString());
```

## 🔀 Git 规范

### 1. 提交信息规范

**格式**: `<type>: <subject>`

**Type 类型**:
- `feat`: 新功能
- `fix`: 修复 bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 重构
- `test`: 测试相关
- `chore`: 构建/工具/配置

**示例**:
```bash
# ✅ 正确
git commit -m "feat: 添加商品搜索功能"
git commit -m "fix: 修复登录跳转问题"
git commit -m "docs: 更新项目文档"
git commit -m "refactor: 优化商品列表加载逻辑"

# ❌ 错误
git commit -m "更新代码"
git commit -m "修复 bug"
git commit -m "update"
```

### 2. 分支管理

**分支命名**:
```bash
main          # 主分支（稳定版本）
dev           # 开发分支
feature/*     # 功能分支（如：feature/product-search）
fix/*         # 修复分支（如：fix/login-bug）
hotfix/*      # 紧急修复分支
```

**分支操作流程**:
```bash
# 创建功能分支
git checkout -b feature/product-search

# 开发完成后合并到 dev
git checkout dev
git merge feature/product-search

# 推送到远程
git push origin dev
```

### 3. 代码提交流程

```bash
# 1. 拉取最新代码
git pull origin dev

# 2. 创建功能分支
git checkout -b feature/xxx

# 3. 开发并提交
git add .
git commit -m "feat: xxx"

# 4. 推送分支
git push origin feature/xxx

# 5. 创建 Pull Request
# 6. 代码审查
# 7. 合并到 dev 分支
```

## 🗄️ 数据库规范

### 1. 表命名

**业务表**: `biz_前缀`
```sql
-- ✅ 正确
biz_product      -- 商品表
biz_order        -- 订单表
biz_category     -- 分类表

-- ❌ 错误
product
t_product
```

**系统表**: `sys_前缀`
```sql
-- ✅ 正确
sys_user         -- 用户表
sys_role         -- 角色表
sys_menu         -- 菜单表
```

### 2. 字段命名

**通用字段**:
```sql
-- 主键
product_id       -- 商品 ID
user_id          -- 用户 ID

-- 通用字段
create_time      -- 创建时间
update_time      -- 更新时间
del_flag         -- 删除标志
create_by        -- 创建人
update_by        -- 更新人

-- 状态字段
status           -- 状态（0 审核中 1 在售 2 已售出）
```

**字段类型**:
```sql
-- 主键：bigint
product_id bigint(20) NOT NULL

-- 金额：decimal
current_price decimal(10,2) DEFAULT NULL

-- 状态：char
status char(1) DEFAULT '0'

-- 时间：datetime
create_time datetime DEFAULT NULL

-- 文本：varchar/text
product_name varchar(200) DEFAULT NULL
product_desc text
```

### 3. SQL 编写规范

**查询语句**:
```sql
-- ✅ 正确
SELECT 
    t1.product_id,
    t1.product_name,
    t1.current_price,
    t2.user_name
FROM biz_product t1
LEFT JOIN sys_user t2 ON t1.user_id = t2.user_id
WHERE t1.del_flag = '0'
  AND t1.status = '1'
ORDER BY t1.create_time DESC
LIMIT #{pageNum}, #{pageSize}

-- ❌ 错误
SELECT * FROM biz_product WHERE status = '1'
```

**索引使用**:
```sql
-- ✅ 正确：为常用查询条件创建索引
CREATE INDEX idx_product_status ON biz_product(status);
CREATE INDEX idx_product_category ON biz_product(category_id);
CREATE INDEX idx_product_user ON biz_product(user_id);

-- ❌ 错误：过多索引影响写入性能
```

## 🔒 安全规范

### 1. 密码安全

**加密存储**:
```java
// ✅ 正确：使用 BCrypt 加密
String encryptPassword = BCrypt.hashpw(password);

// ❌ 错误：明文存储
String password = "123456";
```

**密码校验**:
```java
// ✅ 正确
boolean matches = BCrypt.checkpw(password, encryptPassword);
```

### 2. 权限控制

**方法级权限**:
```java
// ✅ 正确
@PreAuthorize("@ss.hasPermi('biz:product:list')")
@GetMapping("/list")
public TableDataInfo list(BizProduct bizProduct) {
    // ...
}

// ❌ 错误：无权限控制
@GetMapping("/list")
public TableDataInfo list(BizProduct bizProduct) {
    // ...
}
```

**数据权限**:
```java
// ✅ 正确：只查询当前用户的数据
public List<BizProduct> selectMyProducts(Long userId) {
    // 添加 userId 条件
}

// ❌ 错误：查询所有数据
public List<BizProduct> selectAllProducts() {
    // 无用户限制
}
```

### 3. XSS 防护

**输入过滤**:
```java

@Size(min = 0, max = 50, message = "商品名称长度不能超过 50 个字符")
```

## 🎯 最佳实践

### 1. 性能优化

**数据库优化**:
```java
// ✅ 正确：使用分页查询
PageHelper.startPage(pageNum, pageSize);
List<BizProduct> list = bizProductMapper.selectBizProductList(bizProduct);

// ❌ 错误：一次性查询所有数据
List<BizProduct> list = bizProductMapper.selectBizProductList(null);
```

**缓存使用**:
```java
// ✅ 正确：缓存热点数据
@Cacheable(cacheNames = "product", key = "#productId")
public BizProduct getProductById(Long productId) {
    return productMapper.selectBizProductByProductId(productId);
}

// ❌ 错误：每次都查询数据库
public BizProduct getProductById(Long productId) {
    return productMapper.selectBizProductByProductId(productId);
}
```

### 2. 代码复用

**工具类提取**:
```java
// ✅ 正确：抽取公共方法
public class ProductUtils {
    public static String formatPrice(BigDecimal price) {
        return "¥" + price.setScale(2, RoundingMode.HALF_UP);
    }
}

// ❌ 错误：重复代码
String price = "¥" + product.getPrice().setScale(2, RoundingMode.HALF_UP);
```

### 3. 错误处理

**全局异常处理**:
```java
// ✅ 正确：使用全局异常处理器
@RestControllerAdvice
public class GlobalExceptionHandler {
    
    @ExceptionHandler(ServiceException.class)
    public AjaxResult handleServiceException(ServiceException e) {
        return error(e.getMessage());
    }
    
    @ExceptionHandler(Exception.class)

        log.error("系统异常", e);
        return error("系统异常");
    }

```

### 4. 日志记录
